Breakable

fun Breakable(hardness: Double? = null, toolCategories: Set<ToolCategory> = emptySet(), toolTier: ToolTier? = null, requiresToolForDrops: Boolean? = null, breakParticles: Material? = null, showBreakAnimation: Boolean = true): BlockBehaviorFactory<Breakable>

Creates a factory for Breakable behaviors using the given values, if not specified otherwise in the block's config.

Parameters

hardness

The hardness of the block. Used when hardness is not specified in the config, or null to require the presence of a config entry.

toolCategories

The ToolCategories required to break the block. Used when tool_categories is not specified in the config.

toolTier

The ToolTier required to break the block. Can be null to not require a specific tool tier. Used when tool_tier is not specified in the config.

requiresToolForDrops

Whether the block requires a tool to drop its item. Used when requires_tool_for_drops is not specified in the config, or null to require the presence of a config entry.

breakParticles

The type of break particles to spawn in case the block is entity-backed or model-less with no vanilla particles. Can be null. Used when break_particles is not specified in the config.

showBreakAnimation

Whether the break animation should be shown. Used when show_break_animation is not specified in the config.
